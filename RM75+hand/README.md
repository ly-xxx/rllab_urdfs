# RM75-B with RH56DFTP Dexterous Hand

这个文件夹包含了 RM75-B 机械臂与 RH56DFTP 灵巧手的完整组合版 URDF 模型。

## 文件结构

```
RM75B_with_dexterous_hand/
├── urdf/
│   └── RM75B_with_dexterous_hand.urdf  # 组合版主 URDF 文件
├── meshes/
│   ├── RM75-B/                         # RM75-B 机械臂的网格文件
│   │   ├── base_link.STL               # 基座
│   │   ├── link_1.STL to link_7.STL    # 关节链接
│   └── dexterous_hand/                  # RH56DFTP 灵巧手的网格文件
└── README.md                            # 本说明文档
```

## 技术规格

### RM75-B 机械臂
- **自由度**: 7 DOF
- **关节**: joint_1 至 joint_7
- **末端执行器接口**: link_7
- **工作空间**: 标准 RM75-B 工作空间
- **材质**: 白色 (机身), 银色 (法兰)

### RH56DFTP 灵巧手
- **手指数**: 5 指 (拇指、食指、中指、无名指、小指)
- **关节数**: 
  - 拇指: 4 个关节 (right_thumb_1 至 right_thumb_4)
  - 其他四指: 各 2 个关节
- **传感器**: 集成力传感器 (force_sensor)
- **驱动方式**: 联动机构 (使用 mimic 标签)

## 连接方式

RM75-B 的末端法兰 (link_7) 通过 **hand_mount_joint** (fixed joint) 连接到灵巧手的基座 (hand_base_link)。

- **连接类型**: 固定连接 (fixed joint)
- **机械接口**: Φ31.5mm 凸台/凹槽配合
- **安装原点**: `<origin rpy="0 0 0" xyz="0 0 0"/>`

> **注意**: 根据实际安装情况，您可能需要调整 `hand_mount_joint` 的 `origin` 参数来匹配实际的机械安装位置。

## 使用方法

### ROS/Gazebo 中使用

```xml
<include file="$(find xarm6_with_dexterous_hand)/urdf/xarm6_with_dexterous_hand.urdf"/>
```

### Python (PyBullet/MuJoCo/ManiSkill)

```python
import pybullet as p

# 加载 URDF
robot_id = p.loadURDF(
    "path/to/RM75B_with_dexterous_hand/urdf/RM75B_with_dexterous_hand.urdf",
    useFixedBase=True
)
```

## 自定义调整

### 调整灵巧手安装位置

如果需要调整灵巧手相对于机械臂末端的位置和姿态，请修改 `hand_mount_joint` 的 origin 参数:

```xml
<joint name="hand_mount_joint" type="fixed">
    <parent link="link_7"/>
    <child link="hand_base_link"/>
    <!-- 调整 xyz 和 rpy 参数 -->
    <origin rpy="0 0 1.5708" xyz="0 0 0.05"/>
</joint>
```

### 修改材质

所有 link 都定义了 material 属性，您可以根据需要修改材质定义:

```xml
<material name="CustomColor">
    <color rgba="R G B A"/>
</material>
```

## 故障排除

### 问题: Mesh 文件找不到

确保 ROS package path 或 Python 的资源路径正确设置，并且 `package://xarm6_with_dexterous_hand/` 能够正确解析。

### 问题: 关节角度异常

检查关节限位是否正确，以及初始关节角度是否在限位范围内。

### 问题: 碰撞检测问题

如果需要更精确的碰撞检测，可以考虑使用更细致的碰撞网格或调整碰撞参数。
